# Hello World

My first website! A clean, responsive static website built with HTML and CSS.

## Features

- ✨ Modern and clean design
- 📱 Fully responsive
- ⚡ Fast loading
- 🎨 Beautiful gradient colors
- 🔗 Easy to customize

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize the content to make it your own

## Deploying to GitHub Pages

1. Push this repository to GitHub
2. Go to Settings → Pages
3. Under "Source", select main branch
4. Your site will be live at `https://yourusername.github.io/hello-world`

## Customization

- Edit the text in `index.html`
- Change colors in `style.css` (look for the `:root` section)
- Update social links in the contact section
- Replace email with your actual email address
